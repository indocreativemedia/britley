=== Britley ===

== Description ==

Britley, anagram of Liberty, is a clean simple responsive ClassicPress theme based on Bootstrap framework.

== Theme Info ==
* Author: IndoCreativeMedia
* Requires PHP: 7.4
* Requires CP: 1.4
